# Практическая 5–6: REST API + Swagger

Express-сервер для интернет-магазина электроники с документацией OpenAPI 3.0 через Swagger.

## Стек

- **Node.js** + **Express** — HTTP-сервер
- **nanoid** — генерация уникальных ID
- **cors** — настройка CORS для фронтенда
- **swagger-jsdoc** — генерация OpenAPI-спецификации из JSDoc-комментариев
- **swagger-ui-express** — интерактивная документация в браузере

## Запуск

```bash
# установить зависимости (если ещё не установлены)
npm install

# запустить сервер
npm start
# или
node app.js
```

Сервер запускается на **http://localhost:3200**

## Swagger UI

После запуска сервера документация доступна по адресу:

**http://localhost:3200/api-docs**

Там можно просматривать все эндпоинты и отправлять запросы прямо из браузера.

JSON-спецификация: **http://localhost:3200/api-docs.json**

## Эндпоинты API

| Метод    | URL                    | Описание                        |
|----------|------------------------|---------------------------------|
| `GET`    | `/api/products`        | Получить список всех товаров    |
| `POST`   | `/api/products`        | Создать новый товар             |
| `GET`    | `/api/products/:id`    | Получить товар по ID            |
| `PATCH`  | `/api/products/:id`    | Частично обновить товар         |
| `DELETE` | `/api/products/:id`    | Удалить товар                   |

## Модель товара

```json
{
  "id": "abc123",
  "name": "Ноутбук ASUS VivoBook 15",
  "category": "Ноутбуки",
  "description": "Тонкий и лёгкий ноутбук...",
  "price": 64990,
  "stock": 12,
  "rating": 4.7
}
```

| Поле          | Тип      | Обязательное | Описание                        |
|---------------|----------|--------------|---------------------------------|
| `id`          | string   | авто         | Уникальный ID (nanoid 6 симв.)  |
| `name`        | string   | да           | Название товара                 |
| `category`    | string   | да           | Категория                       |
| `description` | string   | да           | Описание                        |
| `price`       | number   | да           | Цена в рублях (≥ 0)             |
| `stock`       | integer  | да           | Количество на складе (≥ 0)      |
| `rating`      | number   | нет          | Рейтинг 0–5 (или null)          |
