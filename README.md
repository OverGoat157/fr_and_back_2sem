# Фронтенд и бэкенд — 2 семестр

Практические работы по курсу «Фронтенд и бэкенд разработка».

## Содержание

| Папка     | Тема                                           |
|-----------|------------------------------------------------|
| `1praka`  | HTML + SCSS — карточка товара                  |
| `2praka`  | Node.js + Express — базовый HTTP-сервер        |
| `3praka`  | Express — маршрутизация и middleware            |
| `4praka`  | REST API + React-фронтенд (интернет-магазин)   |
| `5-6praka`| REST API + Swagger / OpenAPI 3.0               |

## Практическая 4 — интернет-магазин

Полноценное приложение с бэкендом на Express и фронтендом на React.

**Запуск бэкенда:**
```bash
cd 4praka
node app.js   # http://localhost:3000
```

**Запуск фронтенда:**
```bash
cd 4praka/client
npm install
npm run dev   # http://localhost:3001
```

## Практическая 5–6 — Swagger

Тот же REST API, дополненный документацией OpenAPI 3.0.

**Запуск:**
```bash
cd "5-6praka"
npm install
npm start     # http://localhost:3200
```

**Swagger UI:** http://localhost:3200/api-docs

Подробнее — в [5-6praka/README.md](5-6prakа%20/README.md).

## Технологии

- **HTML / SCSS**
- **Node.js / Express**
- **React 18 / Vite**
- **axios**
- **swagger-jsdoc / swagger-ui-express**
